# 확대율 범위 확장 설계

## 목표

화면별 수동 확대율 범위를 기존 50~150%에서 10~200%로 확장함.

## 변경

- 관리 UI 슬라이더 범위: 10~200%
- 슬라이더 조절 간격: 기존 5% 유지
- Rust 설정 정규화 범위: 0.1~2.0
- 기본 확대율: 기존 100% 유지
- OS 화면 배율을 반영하는 최종 WebView 줌 공식은 변경하지 않음

## 데이터 처리

- UI 값은 기존처럼 100으로 나누어 설정에 저장함
- 저장값이 0.1 미만이면 0.1, 2.0 초과이면 2.0으로 정규화함
- 기존 50~150% 설정은 그대로 유지됨

## 검증

- UI 슬라이더의 최소·최대·간격 회귀 테스트
- Rust 정규화의 하한·상한 회귀 테스트
- 전체 Renderer/Rust 테스트, 타입 검사, Clippy, 프로덕션 빌드 실행
